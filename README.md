# C# Técnicas de Programação II
<p>Repositório contendo exercícios das aulas de Técnicas de Programação II da Fatec Jahu.</p>

## Criando uma solução
```
dotnet new sln -n primeiraSolucao
```

## Criando um projeto
```
dotnet new console -n primeiroProjeto
```

## Juntando solução e projeto
```
dotnet sln primeiraSolucao.sln add primeiroProjeto/primeiroProjeto.csproj
```
